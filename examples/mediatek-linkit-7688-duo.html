MediaTek LinkIt Smart 7688 Duo

Note: This documentation is not complete at the moment. This page will be updated once further documentation is available.

This application allows you to turn on and off the LED connected to the pin 13 on the MediaTek LinkIt 7688 Duo using a network connection.

Source code

You can browse the source code for this example at the Evothings GitHub repository

The file index.html contains both the HTML markup and the business logic.

What you need

This example runs in Evothings Viewer on Android or iOS.

  • A MediaTek LinkIt Smart 7688 Duo
  • An iOS or Android smartphone

How to get up and running

Follow these steps to get started with this example:

  1. Run the example named MediaTek LinkIt 7688 Duo from your Evothings Workbench.
  2. Follow this tutorial to properly configure your development board.
